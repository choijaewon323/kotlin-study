# 이펙티브 코틀린 스터디 - 그룹 위치 공유

## 요구사항
- 사용자는 그룹을 생성하거나 방번호로 참여할 수 있다.
- 사용자는 여러 그룹에 속할 수 있다.
- 그룹 생성 시 최대 참여 인원을 설정할 수 있다. (예: 최대 10명)
- 새로운 참여자 추가 혹은 퇴장 시 기존 그룹에 속한 인원들에게 SMS 알림을 발송할 수 있다.
- 사용자는 자신의 현재 위치(위도/경도)를 업데이트할 수 있다.
- 참여자는 그룹 내 다른 인원들의 위치를 조회할 수 있다.
- 그룹 생성자는 그룹을 삭제할 수 있다.
- 참여자는 방을 퇴장할 수 있다.

## API 목록
- 그룹 생성 API v
- 소속된 그룹 목록 조회 API v
- 그룹 참여 API v
- 그룹 위치 조회 API
- 그룹 위치 업데이트 API
- (참여자) 그룹 퇴장 API
- 그룹 삭제 API

## SMS 서버 (localhost:8090)
```javascript
POST /sms/send
{
  "from": "010-0000-0000", // 발신 번호 (서비스 대표 번호, 고정값)
  "to": "010-1234-5678", // 수신 번호 (그룹원 개인)
  "content": "OOO님이 그룹[그룹명]에 참여했습니다." // 메시지 내용
}

```
